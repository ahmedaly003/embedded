# Autonomous BLE Maze-Solving Robot (ECNG 4504)

Embedded firmware + Python PC app for a robot that learns a black-line maze during training, computes the optimal path, then executes it. BLE is used for control and live status reporting.

## Repo Structure
- firmware/     MCU code (C/C++)
- python_app/   PC-side app (Python)
- docs/         diagrams (FSM + architecture)
- hardware/     schematics + PCB files
